ClawdOS Mini App SDK

Build and deploy mini applications for the ClawdOS ecosystem.

Quick Start

Add the SDK to your HTML app:

<!-- Add before closing </body> tag -->
<script src="https://clawdos.com/sdk/clawd-miniapp-sdk.js"></script>
<script>
  // Signal that your app is ready
  clawd.ready();

  // Get user context
  clawd.getContext().then(ctx => {
    console.log('Wallet:', ctx.user.wallet);
  });
</script>

API Reference

clawd.ready()              // Signal app loaded
clawd.close()              // Close the window
clawd.getContext()         // Get user info (Promise)
clawd.onContextUpdate(cb)  // Listen for changes
clawd.setTitle(title)      // Update window title
clawd.openUrl(url)         // Open external URL
clawd.showToast(msg)       // Show notification

Manifest File

Create /.well-known/clawd.json at your app's root for auto-discovery:

{
  "miniapp": {
    "version": "1",
    "name": "My App",
    "iconUrl": "https://myapp.com/icon.png",
    "homeUrl": "https://myapp.com",
    "description": "A cool mini app",
    "developer": {
      "name": "Developer Name"
    }
  }
}
